# Designing dashboards people actually use

This draft explains how to improve dashboard adoption by aligning layouts with real decisions.

## Core principle

Start from the decision sequence, not from available charts.

## Practical checklist

- Define audience-specific questions first.
- Put diagnostic views below summary KPIs.
- Track usage and prune low-value pages.
